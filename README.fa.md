# GetSignalz — ایجنت معاملاتی Hyperliquid

**کانال تلگرام:** [@GetSignalz](https://t.me/GetSignalz)  
**صرافی:** Hyperliquid (قراردادهای دائمی)

---

## معرفی

یک ایجنت هوش مصنوعی معاملاتی که به صورت شبانه‌روزی روی Hyperliquid اجرا می‌شود، سیگنال‌های زنده را در کانال عمومی تلگرام منتشر می‌کند، و هر شب استراتژی و کد خودش را بر اساس داده‌های عملکرد بازنویسی می‌کند.

**ویژگی‌های اصلی:**
- ورود، حد ضرر و حد سود خودکار روی Hyperliquid
- پست‌های سیگنال زنده در تلگرام با به‌روزرسانی P&L
- حداکثر ضرر اهرمی ۲۵٪ در هر معامله (اعمال‌شده در کد)
- یادگیری شبانه: ایجنت تمام معاملات را بررسی و پارامترها و کد خودش را بازنویسی می‌کند
- حد ضرر متحرک برای قفل کردن سود
- محافظت در برابر همبستگی: دو پوزیشن همبسته به صورت همزمان باز نمی‌شوند

---

## معماری

```
live.py          — حلقه اصلی، کنترل ساعات معاملاتی، trailing stop، تریگر شبانه
trader.py        — تولید سیگنال، امتیازدهی اندیکاتور، اندازه‌گیری پوزیشن
executor.py      — wrapper اس‌دی‌کی Hyperliquid، ثبت سفارش
tracker.py       — مانیتور پوزیشن‌های باز، تشخیص بسته‌شدن، محاسبه P&L
tg.py            — فرمت‌بندی و ارسال پیام تلگرام
journal.py       — ذخیره‌سازی تاریخچه معاملات
review.py        — بررسی شبانه/هفتگی + تنظیم‌گر ۸ پارامتر
ai_brain.py      — بهبود شبانه کد و استراتژی با هوش مصنوعی
analyze.py       — تحلیل داده‌های تاریخی
indicators.py    — اندیکاتورهای تکنیکال (RSI، MACD، ADX، BB، EMA، حجم)
backtest.py      — موتور بک‌تست
```

### ۱. پیش‌نیازها

</div>

```bash
pip install hyperliquid-python-sdk eth-account loguru requests
```

<div dir="rtl">

### ۲. پیکربندی

</div>

```bash
cp config.example.py config.py
# فایل config.py را ویرایش کنید و کلیدهای خود را وارد کنید
```

<div dir="rtl">

فایل `config.py` در gitignore قرار دارد و هرگز commit نمی‌شود.

</div>

```python
# config.py
HYPERLIQUID_PRIVATE_KEY = "0xKEY_شما"
HYPERLIQUID_ACCOUNT     = "0xADDRESS_شما"
USE_TESTNET             = True   # برای mainnet مقدار False قرار دهید

TELEGRAM_TOKEN          = "TOKEN_ربات_شما"
TELEGRAM_CHANNEL        = "@کانال_شما"
TELEGRAM_OWNER_ID       = "ID_تلگرام_شما"
```

<div dir="rtl">

### ۳. اجرا

</div>

```bash
python live.py
```

## یادگیری شبانه

هر شب ساعت ۲۳:۰۰ UTC ایجنت:

1. تمام معاملات ۷ روز گذشته را مرور می‌کند
2. ۸ قانون را برای تنظیم پارامترهای اصلی اجرا می‌کند (آستانه امتیاز، ADX، نسبت TP، ریسک درصدی، ساعات معاملاتی، watchlist)
3. کد کامل خودش و داده‌های معاملاتی را به یک مدل هوش مصنوعی می‌فرستد
4. هر تغییر کد پیشنهادی را قبل از اعمال اعتبارسنجی می‌کند (بررسی سینتکس، عدم تغییر اسرار)
5. ساعت ۰۴:۰۰ UTC یک نسخه جدید با changelog کامل به GitHub پوش می‌کند
6. در صورت نبود پوزیشن باز، خودش را ری‌استارت می‌کند
